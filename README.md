# YMM4 Docking Manager

ゆっくりムービーメーカー4（YMM4）向けのツールプラグインです。外部アプリケーションのウィンドウを YMM4 内のドッキングパネルに埋め込み、最大 10 パネルまで一元管理できます。

## できること

- **Docking Manager** パネルで、各ドックの状態の一覧・一括解除・自動復元のトリガー
- **Dock 01〜10** パネルに、任意の外部アプリを **Ctrl + タイトルバードラッグ** でアタッチ
- アタッチ情報の保存と、次回起動時の **自動復元**（対象アプリが起動していれば再アタッチを試行）
- YMM4 終了時などに外部ウィンドウを **デタッチして取り残さない** ための処理
- アタッチ時は **アプリ名（プロセス名など）をタブ／パネル表示名** として利用（未アタッチ時は `Dock 01` 形式）

## 動作環境

- Windows（YMM4 が動作する環境）
- YMM4（プラグイン読み込みに対応した版）

## インストール（通常ユーザー）

1. **`YMM4DockingManager.ymme` を入手**する（リリースから入手）。
2. 次のいずれかで YMM4 に導入する。
   - `.ymme` を**ダブルクリック**して実行する（YMM4 が関連付けされていればインストール処理が走ります）
   - `.ymme` を **YMM4 のウィンドウへドラッグ＆ドロップ**する
3. YMM4 を起動（または再起動）し、ツールメニューに **Docking Manager** および **Dock 01〜10** が表示されることを確認します。

## 基本的な使い方

### 1. ドックパネルを開く

ツールから **Dock 01**〜**Dock 10** のいずれかを開きます。ドックは常に 10 パネル利用できます。

### 2. 外部アプリをアタッチする

1. アタッチしたいアプリを起動したままにします。
2. そのウィンドウの **タイトルバーをドラッグ** し、**Ctrl キーを押したまま**、目的の **Dock パネル内の表示領域** の上でマウスを離します。
3. パネル内にそのウィンドウが埋め込まれ、タブ名などにアプリ名が表示されます。

誤操作防止のため、**Ctrl を押していない場合はアタッチしません**。

アタッチ時に内部で何が起きているか（WinEvent、ヒットテスト、埋め込みの概要）は、[docs/ATTACH_GUIDE.md](docs/ATTACH_GUIDE.md) を参照してください。

### 3. 解除する

- 各 **Dock** パネル上部の「解除」ボタン
- **Docking Manager** の一覧から該当行の「解除」、または「全解除」

解除すると、外部ウィンドウは独立ウィンドウとして元に戻るよう処理します。

### 4. 自動復元

アタッチ状態に関する **復元用のヒント**（プロセス名・実行ファイルパスなど）は、ユーザーのドキュメント配下に JSON で保存されます。次回 YMM4 起動後、**Docking Manager** の「自動復元」で、保存された内容に基づき再アタッチを試みます。対象アプリが起動していない場合はアタッチされません。

**保存場所の例:** `ドキュメント\YMM4 Docking Manager\settings.json`

## 制限・注意事項

- **Windows 標準のエクスプローラー**は、埋め込み時の相性問題があるためアタッチを拒否します。
- 外部アプリ側の実装によっては、親ウィンドウ変更やスタイル変更に追従できず、表示や入力が不安定になる場合があります。
- 本プラグインは **利用者自身の責任**でご利用ください。データ損失やアプリの異常終了などについて、作者は一切の保証をしません。
